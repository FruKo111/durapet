================================================================================
HOSTİNGER — Ortam değişkenleri (Next.js panel: durapet.com.tr)
================================================================================
Panelde: Web uygulaması / Dağıtım → Ortam değişkenleri (Environment variables)
Her satır = bir değişken. Kaydettikten sonra mutlaka YENİDEN DEPLOY / BUILD al.

Aşağıdaki BOŞLUKLARI doldur. Sonunda / kullanma.

--------------------------------------------------------------------------------
ZORUNLU
--------------------------------------------------------------------------------

NEXT_PUBLIC_API_BASE_URL
  → Node API’nin internetten açık HTTPS adresi.
  → Örnek: https://api.durapet.com.tr
  → API henüz yoksa önce API’yi bir sunucuda yayınla; panel tek başına çalışmaz.

NEXT_PUBLIC_SUPABASE_URL
  → Supabase Dashboard → Project Settings → API → Project URL
  → Örnek: https://xxxx.supabase.co

NEXT_PUBLIC_SUPABASE_PUBLISHABLE_DEFAULT_KEY
  → Supabase → API → "publishable" / "anon" public key (sb_publishable_... veya eyJ... ile başlayan anon JWT)
  → Service role key ASLA koyma (sadece sunucuda kalır).

--------------------------------------------------------------------------------
ÖNERİLEN (QR ve kimlik linkleri için)
--------------------------------------------------------------------------------

NEXT_PUBLIC_SITE_URL
  → Canlı panel adresin, örn: https://durapet.com.tr
  → www kullanıyorsan: https://www.durapet.com.tr

NEXT_PUBLIC_QR_PUBLIC_BASE_URL
  → Genelde SITE_URL ile aynı (QR’da açılacak site kökü).

--------------------------------------------------------------------------------
OPSİYONEL
--------------------------------------------------------------------------------

NEXT_PUBLIC_GOOGLE_MAPS_API_KEY
  → Sadece panelde harita önizlemesi kullanıyorsan (Maps Embed API + referrer kısıtı).
